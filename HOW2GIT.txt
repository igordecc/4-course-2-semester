HOW2 сделать что то простое в системе контроля версий Git

git  add --all - добавить все untraked изменения в локальную ветку
git commit - commit в локальную ветку
	в открутом окне нажать английскую "a" для вставке
или
git commit -a -m "commit commentary 123 abc" - добавить всё + сразу закомитить с коментарием
 

git push - push в текущую ветку
git push origin master - push в мастер ветку


git status - проверить в какой ты ветке и посмотреть последние изменения
git checkout - посмотреть последние изменения
git fetch && git rebase origin/master - применить изменения локальной ветки к удалённой и перебазировать локальную в origin/master (превратить её из форка в основную ветку)
